    How do the website settings work?

    You can pick a dark or light website theme, whichever you prefer. This changes the colors of all pages, code highlighting, and some screenshots. The default auto theme picks dark or light based on your browser and system settings.

    If bold & italic code is enabled font styles are used when highlighting certain code fragments. For example comments and functions will be italic and keywords will be bold.

    If code ligatures are enabled certain code symbol combinations will be visually merged. For example, -> becomes -> and == becomes ==. This is a feature of the JetBrains Mono font.

    If Visible code tabs is enabled code tabs will be visualized with vertical lines, one per indentation level.

    These settings are saved in your browser's local storage.

    Why text and not video tutorials?

    As my tutorials focus on programming they are mostly about writing code, so it seems natural to use text. Understanding the code might require a lot of jumping back and forth, pausing, looking things up, experimenting, and so forth, all at your own pace. I think text is more convenient than video for this.
